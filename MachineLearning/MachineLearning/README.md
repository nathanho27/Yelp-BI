# Machine Learning Models (Yelp-BI)

This folder contains all ML workflows, notebooks, and exported model artifacts.

## Models

### 1. Expected Rating Model
Predicts expected Yelp rating based on:
- price level  
- review count  
- categories  
- competition density  
- city-level patterns  

### 2. Closure Risk Model
Predicts the probability that a business will close.
